msec : 타이머 작동에서 실제 msec를 의미함
sec : 실제 sec와는 다르며 1이 될 때 게임 갱신이 됨
st : 처음엔 0이며 타이틀 화면에서 수신을 받아 1이되고, 게임 오버 시에 0이 됨
level : 게임 레벨 변수
lines : 없앤 라인의 수. 10이되면 레벨이 오르고 0으로 초기화됨
score0~5 : 각각 자리수의 점수를 표현하기 위한 변수
bDrawBlock : 새로운 블록을 그리기 위한 변수
bPause : 정지 상태를 나타내기 위한 변수
a[25][12]	 : 배경이 되는 2차원 배열로 각 위치의 숫자에 따라 화면이 출력됨
i : 배열 출력 또는 확인 시 y축의 좌표로 사용되는 변수
j : 배열 출력 또는 확인 시 x축의 좌표로 사용되는 변수
bx : 현재 블록의 x축 좌표 
by : 현재 블록의 y축 좌표 (가장 위쪽이 0)
L[8] : 레벨에 따른 LED 출력 패턴 
N[10] : 7 Segment 숫자 출력 패턴
blocks[7][4][4][4] : 4차원 배열로 타입, 회전에 따른 블록들이 저장되어 있음
type : 현재 블록의 타입
ntype : 다음 블록의 타입
rotate : 현재 블록의 회전 상태
seed : 랜덤으로 블록을 고르기 위한 시드 변수로, 타임 인터럽트와 수신 인터럽트 발생시에 변화됨
